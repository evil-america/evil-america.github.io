# 2008年金融危机
## 事件经过
### 2007年4月
美国第二大的次级房贷公司破产.
### 2008年8月
次贷危机爆发。美国最大的两家房贷企业股价暴跌，投资者及机构亏损严重；
### 2008年9月15号
雷曼兄弟申请美国史上最大破产保护，债务超过6130亿美元，全球股市暴跌。
