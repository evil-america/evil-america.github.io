# 如何应对封锁和制裁，只能坐以待毙吗？
## 事件经过
### 2018年3月22日
美国总统特朗普在白宫签署了对中国输美产品征收关税的总统备忘录。

### 北京时间6月15日晚
美国政府发布了加征关税的商品清单，将对从中国进口的约500亿美元商品加征25%的关税。

### 2019年5月9日
美国政府宣布，自2019年5月10日起，对从中国进口的2000亿美元清单商品加征的关税税率由10%提高到25%。
