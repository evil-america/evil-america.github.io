# 维基解密事件，曝光美军屠杀阿富汗平民
## 事件经过
### 2010.4
维基解密发放了美军在2007年于巴格达滥杀平民的片段。
### 2010.7.25
维基解密网站 (Wikileaks) 通过英国《卫报》、 德国《明镜》和美国《纽约时报》公布了92000份美军有关阿富汗战争的军事机密文件。机密文件中，最具破坏性的消息是北约联军在阿富汗杀死平民的事件。
