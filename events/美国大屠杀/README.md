# 美国大屠杀
## 事件经过
### 16世到19世纪末
美洲大屠杀又称印第安人大屠杀，是一场惨绝人寰的种族大屠杀，大约死亡人数2500万人。

### 1783年
美国独立以后，对印第安人的屠杀和虐待并没有丝毫收敛。随着资本主义的迅速发展，为了开拓疆土，美国政府把印第安人驱逐出祖居地。

### 1830年
美国政府通过《印第安人迁移法案》，规定东部的印第安人要全部迁往密西西比河以西的为他们划定的保留地中去，实行种族隔离和迫害。

### 1924年
美国国会才通过了《印第安人公民资格法》，宣布凡在美国境内出生的人均为美国公民。
