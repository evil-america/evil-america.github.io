# 美国空袭致伊朗将军身亡，震动全球资本市场

## 事件经过
### 2019年1月3日凌晨
伊拉克首都巴格达国际机场遭3枚火箭弹袭击，造成至少8人死亡。美方承认发动袭击。其中，伊朗革命卫队圣城旅将领Qassem Soleimani将军在空袭中丧生。受此影响，国际油价大幅飙升逾4%，金价飙涨。
